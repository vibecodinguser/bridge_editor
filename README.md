# Bridge Editor

Расширение для **Chromium** (Manifest V3) и **Firefox** (сборка MV2), которое отображает редактор [Народной  карты (НЯК)](https://n.maps.yandex.ru/) и [Nakarte.me](https://nakarte.me/): в одной вкладке отображаются оба источника, а движение по карте, масштаб и положение курсора в Яндекс.НК отражаются на Nakarte.

## Возможности

- **Синхронизация вьюпорта** — широта, долгота и зум, извлечённые из URL Яндекс.НК, применяются к карте Nakarte через `history.replaceState` и искусственное событие `hashchange`, чтобы страница Nakarte обновилась так же, как при ручной навигации.
- **Синхронизация курсора** — смещение курсора относительно центра экрана в Яндекс.НК переводится в географические координаты (Web Mercator) и отображается на Nakarte оверлеем (перекрестие).
- **Сохранение слоёв Nakarte** — коды слоёв из хэша `nakarte.me` сохраняются в `chrome.storage.local` и подставляются при следующем открытии split view.
- **Split view** — по клику на иконку расширения открывается окно с двумя `iframe` (слева Яндекс.НК, справа Nakarte), с изменяемой шириной панелей.

## Стек

- WXT
- TypeScript
- Vitest

## Разработка

Требуется **Node.js** (рекомендуется LTS).

```bash
npm install
npm run dev
```

Для отладки в Firefox:

```bash
npm run dev:firefox
```

## Сборка и упаковка

```bash
npm run build
npm run build:firefox
npm run pack
```

Артефакты окажутся в `.output/`.

## Проверка качества

```bash
npm run compile
npm run lint
npm test
```

## Как пользоваться

1. Установите собранное расширение из каталога `.output/chrome-mv3` (или `firefox-mv2`) в режиме разработчика.
2. Нажмите иконку **Bridge Editor** в панели инструментов — откроется split view с картой Яндекса слева и Nakarte справа.
3. Ведите карту в Яндекс.НК — вьюпорт и курсор будут дублироваться на Nakarte (в той же вкладке split view), пока service worker и вкладка активны.
